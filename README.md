## ğŸ Getting Started

### Prerequisites
- Node.js (v16 or higher)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/jorense/safe-practice-exams.git

```

2. Install dependencies:
```bash

3. Start the development server:

```
